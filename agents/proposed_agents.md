# Proposed Agents

Agents are proposal-only in PUBLIC MODE.

## Knowledge Distiller

- Purpose: convert raw notes into summaries, decisions and next actions.
- Inputs: public-safe notes.
- Outputs: structured markdown.
- Limits: no private data, no invented sources.

## Project Router

- Purpose: classify work requests into project contexts.
- Inputs: user request and available public route files.
- Outputs: route recommendation and caveats.
- Limits: no hidden cross-repo access.

## Agent Factory

- Purpose: draft future agent definitions.
- Inputs: desired responsibility and constraints.
- Outputs: proposed agent spec.
- Limits: no live agent creation or tool grants.
