# ULTRON — Work PC Install Guide

## Status
TENTATIVE — Pending work PC access (post May 26)

## Requirements
- Node.js 18+
- Git
- Browser (Chrome recommended for Voice API)

## Installation
git clone https://github.com/Martind-40/tony-rip.git ultron
cd ultron/app
npm install

## Run frontend
npm run dev

## Run backend (separate terminal)
npm run backend

## Access
- Local: http://localhost:5173
- Production: https://tony-rip-orpin.vercel.app

## Security rules
- No .env with real keys committed
- No access to work files outside workspace/
- No git push without Chief approval
- Backend local only
